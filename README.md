# Game Artist Task Manager

Task management tool for game artists — track art tasks, deadlines, and team feedback in one place.

## Features

### Kanban Board
- Drag-and-drop task cards between status columns (Todo, In Progress, Done)
- Inline editing of task details directly on cards
- Visual status indicators with neo-brutalism styling

### Calendar View
- Monthly calendar grid (Sun–Sat) with multi-day task bars
- Task bars color-coded by status: green (done), red (overdue), blue (active), gray (upcoming), purple (feedback)
- Drag tasks between dates to reschedule
- Today highlighted with red circle marker
- Right-click context menu (Edit / Delete)

### Table View
- Sortable, filterable task list with inline editing
- Columns: Name, Est. Time, Dates, Status, Priority
- Expandable rows showing task feedbacks
- Bulk selection and delete

### KPI Dashboard
- Task completion analytics by year/date range
- Export to Excel
- Visual charts for productivity tracking

### Task Management
- Create tasks with name, game, project, estimated time, date range, status, priority
- Feedback system: add feedback entries with their own date ranges to any task
- Smart input parser: natural language task creation
- Auto-archive completed tasks

### Multi-language
- Vietnamese and English interface
- Toggle language from header

### Design
- Neo-brutalism theme: bold borders, hard shadows, monospace typography
- Dark / light mode toggle
- Responsive layout
